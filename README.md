# Alula Code Assignment

Written by Alex Seefeldt

## Additional Features

1. URL restores app state: e.g. visiting https://\<app domain\>/users/edit/12 from an empty tab will load the modal to edit user with ID 12
2. User's full name is displayed in page title when editing
3. Error toast pops up if API requests fail
4. Edit user modal is dismissible with close button or escape key
5. Instantaneous updates!

## Angular stuff

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.0.

### Development server

To start a local development server, run:

```bash
ng serve
```

### Building

To build the project, run:

```bash
ng build
```
